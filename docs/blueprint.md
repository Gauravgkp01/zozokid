# **App Name**: KidTube Safe

## Core Features:

- Login and Signup: Secure authentication system for parents.
- Child Profile Creation: Parents can create profiles for their children.
- Content Filtering: Parents can add specific YouTube channels and content types for their children.
- Safe Feed: Display of YouTube Shorts, curated based on parental settings. The AI tool must carefully avoid content from channels and content types that the parent excluded. It should adapt and become better over time as more information about exclusions become known.
- Parent Dashboard: Centralized dashboard for parents to manage child profiles, content filters, and view analytics.
- Analytics: Track total watch time and category-wise video consumption for each child.
- Settings Management: Allows parents to adjust configurations such as channels and other settings related to safe YouTube Consumption

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to create a calm and friendly environment.
- Background color: Light grey (#F0F4F7) for a clean and unobtrusive backdrop.
- Accent color: Gentle orange (#E59866) to highlight important elements and call-to-action buttons.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern but still friendly look in all text.
- Use child-friendly, playful icons related to video categories and settings.
- Simple and intuitive layout, optimized for both parents and children, with clear visual hierarchy.
- Gentle, engaging animations and transitions to provide a seamless and enjoyable user experience.
